# Floating Chat Button 設計仕様

## 1. 概要
- 目的: ポートフォリオサイト全体でアクセス可能なAIチャットアシスタントへの入り口を提供
- 配置場所: 画面右下に固定配置（全ページ共通）
- 機能: クリックでオーバーレイチャットパネルを表示（将来実装）

## 2. コンポーネント構成
- ファイル: `components/ui/FloatingChatButton.tsx`
- 種別: React Client Component
- 配置: `app/layout.tsx` でグローバルに配置
- 内包:
  - アイコン（MessageCircle）
  - タイトル（"AIアシスタント"）
  - 説明文（"ポートフォリオについて質問"）

## 3. デザイン仕様

### 3.1 ビジュアルデザイン
- 2段構成のカード型ボタン
  - 上段: アイコン + "AIアシスタント"
  - 下段: "ポートフォリオについて質問"
- 背景: プライマリカラーのグラデーション
- テキスト: 白色で視認性確保
- シャドウ: 浮遊感のあるドロップシャドウ

### 3.2 レイアウト
- 位置: `fixed bottom-6 right-6` (24px margin)
- サイズ: 
  - デスクトップ: 幅 200px
  - モバイル: 幅 160px
- z-index: 50（他のコンテンツより上）

### 3.3 インタラクション
- ホバー: 
  - スケール 1.05倍
  - より深いシャドウ
  - カーソル: pointer
- クリック:
  - スケール 0.95倍（押下感）
  - 将来: チャットパネル表示
- アニメーション:
  - 初期表示: フェードイン + スライドアップ
  - トランジション: 0.2秒

## 4. 実装詳細

### 4.1 スタイリング
```tsx
className="fixed bottom-6 right-6 z-50 
  bg-gradient-to-br from-primary to-primary/80 
  text-white rounded-lg shadow-lg 
  hover:scale-105 hover:shadow-xl 
  transition-all duration-200"
```

### 4.2 レスポンシブ対応
- モバイル（640px未満）:
  - サイズ縮小
  - 位置調整: `bottom-4 right-4`
  - フォントサイズ調整

### 4.3 アクセシビリティ
- role="button"
- aria-label="AIアシスタントを開く"
- tabIndex={0}
- キーボード操作対応（Enter/Space）
